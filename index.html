Questions and Answers

First Question:



Category Web 1.0 Web 2.0 Web 3.0
Search Engines Speed&Accuracy Searching engines retrieve macro content. The search is quite fast, however results are often inaccurate Searching engines retrieve tags with micro contents. The tagging process is manual and covers negligible percents of the WWW. Web 2.0 tags everything Searching engines will hopefully retrieve micro content texts which were tagged automatically: translation of billions of Web 1.0 macro contents into micro contents. The result could be a more precise search, since tagging can solve part of the ambiguity that homonyms and synonyms introduce into the process of Search
Consumption Companies publish content that people consume People are able to publish content that other people can consume, companies build platforms that allow people to publish content for other people People build applications that allow people to interact with each other, companies build platforms that let people publish services by leveraging the associations between people or social content
Communication “One-way” communication – pushed web, text/graphics based flash (no interaction between publishers and readers) “Two-way” communication – social networks, blogs and etc allow communication between publishers and readers Potentially a more interactive communication between users (Integrated games, education and business, MUVEs)









Second Question:

Web 1.0


When looking for information about 'side effects of drugs', the user will need specific address of the website he's looking for information at. In addition, while the search would be quite quick, the results might be inaccurate due to the fact the searching engine only retrieves macro content.


Web 3.0


When looking for information about 'side effects of drugs', the user will have to write specific search word/s that relate to the subject ('side effects', 'drugs' and etc). However, the user won't have to know a specific website address in order to perform the query. There's a hope that the search engines would retrieve micro content texts that were tagged by an automatic process. In addition, the search during this web generation would be based on semantics along with other advanced methods, which are meant to make the results more precise than results that past generations would've retrieved. The tagging should be solving part of the ambiguity that homonyms and synonyms introduce into the process of search.
